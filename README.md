# WarixTeam-SimpleNickV0.2

Un plugin simple de nick pour un serveur pocketmine en 4.0.0 et plus.

# Commandes
- /nick -> Changer son pseudo
- /unnick -> Remettre son pseudo à zero
- /resetnick -> (modérateur) Reset le pseudo d'un joueur
